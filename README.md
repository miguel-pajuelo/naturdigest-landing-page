# NaturDigest Landing (GitHub Pages)

Landing page estática para validar interés en NaturDigest.

## Publicar en GitHub Pages (rápido)

1. Crea un repo en GitHub (ej: `naturdigest-landing`)
2. Sube **index.html** y la carpeta **assets/**
3. Settings → Pages → Deploy from a branch → `main` / `/root`
